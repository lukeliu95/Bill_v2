# 日本標準産業分類 - 大分類

## 大分類一覧（A-T）

| コード | 産業名 | 主な内容 |
|--------|--------|----------|
| A | 農業、林業 | 耕種農業、畜産農業、育林業 |
| B | 漁業 | 海面漁業、内水面漁業、水産養殖業 |
| C | 鉱業、採石業、砂利採取業 | 金属鉱業、石炭・亜炭鉱業、採石業 |
| D | 建設業 | 総合工事業、職別工事業、設備工事業 |
| E | 製造業 | 食料品〜機械器具まで全製造業 |
| F | 電気・ガス・熱供給・水道業 | 電気業、ガス業、水道業 |
| G | 情報通信業 | 通信、放送、情報サービス、ソフトウェア |
| H | 運輸業、郵便業 | 鉄道、道路運送、水運、航空、郵便 |
| I | 卸売業、小売業 | 商社、小売、卸売 |
| J | 金融業、保険業 | 銀行、証券、保険、金融商品取引 |
| K | 不動産業、物品賃貸業 | 不動産取引、賃貸、リース |
| L | 学術研究、専門・技術サービス業 | 法律、会計、建設設計、調査、広告 |
| M | 宿泊業、飲食サービス業 | 旅館、ホテル、飲食店 |
| N | 生活関連サービス業、娯楽業 | 洗濯、美容、娯楽施設 |
| O | 教育、学習支援業 | 学校、塾、各種スクール |
| P | 医療、福祉 | 医療機関、介護サービス、社会保険 |
| Q | 複合サービス事業 | 複合的なサービス提供 |
| R | サービス業（他に分類されないもの） | 洗染、修理、廃棄物処理 |
| S | 公務（他に分類されるものを除く） | 国家公務、地方公務 |
| T | 分類不能の産業 | - |
